Pet Food & Services Marketplace

Flutter Marketplace Application

Home / Projects / Pet Food & Services Marketplace

Project Overview

A marketplace mobile application allowing users to book pet services and purchase pet-related products from nearby providers.

My Role

I worked as a Flutter Developer, building the application end-to-end, from UI and architecture to integrations and deployment.

What I Built

  • Service booking and scheduling workflows
  • E-commerce product browsing and checkout
  • Location-based discovery using Google Maps
  • Secure payment gateway integration
  • Real-time notifications and order tracking

Tech Stack

Flutter • Firebase • REST APIs • Google Maps • Payments

Outcome

The application was deployed as a production-ready solution, supporting real users with reliable booking and purchasing flows.
